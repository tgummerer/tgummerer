--- title: SVN Analyzer description: > Graphical SVN anlayzer. --- {% mark excerpt -%} Graphical Subversion analyzer {%- endmark %} A University project creating a graphical analyzer for Subversion repositories. It can display the revision history, general information about the repository, commits over time, total user activity, user activity over time, project activity (commits to the folders in the root path of the repository) and project activity over time. It can also export the data to a CSV file. It's using a hsqldb as backend. Download JAR file: svnanalyzer.jar
